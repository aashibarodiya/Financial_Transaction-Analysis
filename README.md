# Financial Transactions Analysis (Capstone Project)

## 📌 Objective
To analyze financial transaction data using Excel and derive meaningful insights.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP / XLOOKUP
- Data Cleaning Techniques

## 📊 Tasks Performed
- Counted total number of transactions
- Identified and handled missing values
- Removed duplicate records
- Extracted Year from Transaction Date
- Categorized Credit Score
- Created Pivot Tables and Dashboard

## 📈 Key Insights

- Payments & Withdrawals dominate transactions  
- Kotak Bank → Highest Profit  
- South Region → Highest Loan Contribution  
- Credit Card holders → Highest Credit Score  
- Transactions remain stable over time  
- North & West → Growth Opportunity  
- Kolkata → Top Performing City
